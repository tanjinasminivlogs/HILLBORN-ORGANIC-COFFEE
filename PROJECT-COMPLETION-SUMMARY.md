## 🆘 Support & Troubleshooting

### Common Setup Issues

**Images Not Loading**
→ Ensure assets/ folder exists and images are properly named
→ Check file paths in HTML match actual filenames
→ Verify images are in correct folder

**Styles Not Applying**
→ Clear browser cache (Ctrl+Shift+Delete)
→ Verify styles.css link in HTML head
→ Check for CSS typos in console

**JavaScript Not Working**
→ Open DevTools (F12) and check console
→ Verify script.js is linked before closing </body>
→ Check for JavaScript errors

**Form Not Submitting**
→ Check browser console for validation errors
→ Verify API endpoint in handleBooking() function
→ Test form validation in console

### Quick Fixes Checklist
- [ ] Hard refresh browser (Ctrl+F5)
- [ ] Check browser console for errors
- [ ] Verify file paths are correct
- [ ] Check file permissions (644 for files, 755 for folders)
- [ ] Verify internet connection
- [ ] Test in different browser
- [ ] Clear browser cache completely
